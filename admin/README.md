# Admin Panel - TezTecch Buzz

Separate admin application running on **Port 5174**.

## Quick Start

```bash
npm install
npm run dev
```

Admin panel will open at: http://localhost:5174

## Build

```bash
npm run build
```

Output: `dist/` folder

## Environment Variables

Create `.env` file:
```env
VITE_API_URL=http://localhost:5000
```
